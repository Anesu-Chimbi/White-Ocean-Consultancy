## 🌐 Hosting on GitHub Pages

### Step 1: Create a GitHub Account
If you don't have one, sign up at [github.com](https://github.com)

### Step 2: Create a New Repository
1. Click the "+" icon and select "New repository"
2. Name it: `white-ocean-consultancy` (or your preferred name)
3. Add description: "Official website for White Ocean Consultancy"
4. Select "Public" (required for free GitHub Pages)
5. Check "Add a README file"
6. Click "Create repository"

### Step 3: Upload Files
1. Click "Add file" → "Upload files"
2. Upload the `index.html` file
3. Commit the changes

### Step 4: Enable GitHub Pages
1. Go to Settings (top right)
2. Scroll to "Pages" section
3. Under "Source", select "main" branch
4. Select "/" (root) folder
5. Click Save
6. Wait a few minutes for deployment

### Step 5: Access Your Website
Your site will be live at: `https://yourusername.github.io/white-ocean-consultancy/`

## 📝 Customization

### Update Company Information
Open `index.html` and find these sections to update:

```html
<!-- Contact Information -->
<p><strong>Location:</strong> Harare, Zimbabwe</p>

<!-- Contact Alert -->
<button class="btn-primary" onclick="alert('Contact White Ocean Consultancy\n\nLocation: Harare, Zimbabwe\n\nEmail: your-email@whiteocean.co.zw\n\nWe look forward to discussing your HR needs!')">
```

### Add Your Email
Replace the contact alert with actual email/phone when available:

```javascript
// Option: Replace alert with mailto link
<a href="mailto:your-email@whiteocean.co.zw" class="btn-primary">Get in Touch</a>
```

### Change Colors
Edit the CSS variables at the top of the `<style>` section:

```css
:root {
    --primary: #0c447c;           /* Main blue color */
    --primary-light: #185fa5;      /* Lighter blue */
    --accent: #00d4ff;             /* Cyan accent */
    /* ... other colors ... */
}
```
