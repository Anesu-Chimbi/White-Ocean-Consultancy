## ⚡ Quick Deploy (5 minutes)

### 1. Go to GitHub.com and Create Repository
- Click **+** → **New repository**
- Name: `white-ocean-consultancy`
- Make it **PUBLIC**
- Click **Create**

### 2. Upload Files
- Click **Add file** → **Upload files**
- Upload: `index.html`
- Click **Commit changes**

### 3. Enable GitHub Pages
- Go to **Settings**
- Find **Pages** section
- Select `main` branch and `/ (root)`
- Click **Save**

### 4. Live!
Website is now at: `https://yourusername.github.io/white-ocean-consultancy/`
